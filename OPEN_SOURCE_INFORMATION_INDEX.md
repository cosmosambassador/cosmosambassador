## 6. Public Posting Rule

Before anything from the archive becomes public, classify it as one of the following:

1. **Established fact / sourced public history**
2. **Personal memory / first-person testimony**
3. **Research note / working hypothesis**
4. **Symbolic or metaphysical language**
5. **Regression-story or experiencer narrative**
6. **Fiction / worldbuilding / creative treatment**
7. **Operational website or publishing note**

This keeps the archive useful, ethical, and readable.
